## 4. BIOS & Windows Configuration (Summary)

### 4.1 BIOS

- Load XMP/DOCP to run RAM at rated speed (3200‚Äì3600 MHz)
- Leave CPU at stock (no manual OC)
- CPPC / ‚ÄúPreferred Cores‚Äù: Enabled / Auto
- Above 4G decoding: Enabled
- Resizable BAR (SAM): Enabled if available
- Fast Boot / Ultra Fast: Disabled (to keep USB boot / BIOS access easy)

### 4.2 Windows

**Admin account (you):**
- Microsoft account sign‚Äëin
- Full admin rights
- Used for installation / updates

**Arcade account (kids/guests):**
- Local standard user
- Auto‚Äëlogin enabled
- BigBox launched on login
- Limited permissions; cannot install software

**Privacy / Power:**
- Telemetry reduced to minimal
- Tailored experiences / ad ID disabled
- Sleep / hibernation disabled
- Screen saver off

## 5. Software Stack & Scoop Script

### 5.1 Directory Layout

**On C:**
```text
C:\Emulation\Emulators\   # All emulator folders
C:\Emulation\Tools\       # Helper tools, scripts
```

**On External Drive (X:):**
```text
X:\LaunchBox\              # LaunchBox + BigBox (optional)
X:\Emulation\BIOS\        # BIOS per system
X:\Emulation\Games\       # ROMs, ISOs, etc.
```

### 5.2 Scoop‚ÄëBased Emulator Setup

A **Bash script** (for Git Bash / WSL) manages emulator installs via Scoop:

**Responsibilities:**
- Ensure Scoop is installed
- Add buckets (`main`, `extras`, emu‚Äërelated bucket)
- Install emulators:
  - RetroArch
  - Dolphin
  - PCSX2
  - RPCS3
  - Xenia (if available)
  - Ryujinx / Yuzu fork
  - Citra / Lime3DS
  - MAME or FBNeo
  - Supporting CLI tools
- Relocate or link installed apps into:
  - `C:\Emulation\Emulators\<Name>`
- Optionally pin versions
- Provide clear update commands (e.g., `scoop update`)

This script should live in the repo under `/software/emulator_setup.sh` and be kept in sync with actual emulator choices.

### 5.3 LaunchBox / BigBox Configuration

- Import platforms from `X:\Emulation\Games\<Platform>`
- Associate each platform with the correct emulator from `C:\Emulation\Emulators\`
- Configure:
  - Unified button mappings
  - Per‚Äësystem default cores (for RetroArch)
  - Per‚Äësystem startup/shutdown screens
- BigBox:
  - 4:3‚Äëfriendly theme
  - Kiosk mode on
  - Exit + Options locked with password
